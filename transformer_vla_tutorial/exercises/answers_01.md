# 01 章练习答案

## 1. token 数量

每个样本 20 个 token，batch 里 8 个样本，所以总 token 数是 160。每个 token 是 512 维。

## 2. qpos 为什么能和其他 token 放一起

因为 Transformer 只要求每个 token 都是同样 hidden dimension 的向量。通过线性层可以把 qpos 从 7 维映射到 512 维，与语言/图像 token 对齐。

但注意：维度一样不代表语义天然对齐。模型需要通过训练学会这些 token 的含义。

## 3. 行为克隆和人类模仿

相似点：都从示范中学习动作。  
不同点：人类有常识、触觉、因果推理和自我纠错；普通行为克隆通常只是拟合数据分布，出了训练分布容易失败。
