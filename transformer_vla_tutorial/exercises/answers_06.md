# 06 章练习答案

## 1. 为什么要关心动作表示

机器人最终要执行物理动作。动作表示决定了精度、平滑性、频率、误差形式和安全边界。再大的 VLM，如果动作接口不合适，也可能控制不好。

## 2. 离散 action token vs 连续回归

离散 token：容易接入语言模型范式，但有量化误差，平滑性和控制频率需要额外处理。  
连续回归：输出直接，但多模态动作可能被平均，导致动作不自然。

## 3. VLM + diffusion 中 Transformer 的作用

Transformer 主要负责视觉语言理解和条件编码：识别物体、理解指令、融合上下文。Diffusion head 根据这些条件生成连续动作。

## 4. 双臂插拔任务中 ACT 和 VLA 的优势

ACT：擅长从遥操作数据中学习稳定的短期连续动作块，适合精细固定任务。  
VLA：更擅长语言指令、对象变化、语义泛化和多任务迁移，但需要处理精细控制接口。
