# 00 章练习答案

## 1. patch 数量

224 / 16 = 14，所以每边 14 个 patch，总数是：

$$
14 × 14 = 196
$$

如果再加一个 `[CLS]` token，就是 197 个 token。

## 2. 机器人动作向量和文本 token 的区别

文本 token 通常是离散的：词表中的一个 ID。  
机器人动作常常是连续的：每一维是实数，并且直接影响物理世界。

所以机器人动作更强调：

- 平滑性；
- 安全范围；
- 控制频率；
- 误差累积；
- 与低层控制器的接口。

## 3. 动作块为什么更稳定，也可能更危险

更稳定：模型一次规划未来一小段，能减少逐帧预测带来的抖动。  
更危险：如果当前观测误判，未来一整段动作都可能偏离；所以常用 temporal ensembling、闭环重规划和安全限制。
