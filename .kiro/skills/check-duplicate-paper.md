# 检查论文重复工具

## 使用方式

当需要检查一篇论文是否已经被收录时，提供以下信息之一：
- 论文标题（中英文皆可）
- arXiv ID（如 2505.18719）
- 论文简称/缩写（如 DPPO、VLA-RL、TGRPO）

## 检查流程

1. 在 `/home/wahaha/paper/论文综述/` 目录中搜索是否有匹配的文件
2. 在已有文件的 frontmatter、标题、论文标题行中搜索关键词
3. 如果找到匹配 → 返回"已存在"并给出文件路径
4. 如果没找到 → 返回"未收录"，可以新建

## 搜索策略

对输入的论文信息，按以下顺序搜索：
1. 精确匹配 arXiv ID
2. 文件名关键词匹配
3. 文件内容中论文标题匹配
4. 文件内容中方法简称匹配

## 输出格式

```
✅ 已存在：/论文综述/006_VLA_RL_PPO直接训练自回归VLA.md
```

或

```
❌ 未收录：可以新建文章
```
